# Utilisation quotidienne

## Ajouter une entrée

1. Ouvrir le formulaire d'ajout.
2. Renseigner les informations nécessaires (service, identifiant, mot de passe).
3. Ajouter des notes si utile.
4. Enregistrer.

## Modifier une entrée

1. Rechercher l'entrée.
2. Ouvrir l'édition.
3. Mettre à jour les champs.
4. Enregistrer.

## Supprimer une entrée

1. Ouvrir l'entrée concernée.
2. Cliquer sur Supprimer.
3. Confirmer l'action.

## Gérer la sécurité au quotidien

- utiliser la génération de mot de passe quand disponible,
- éviter la réutilisation de mots de passe,
- surveiller les alertes et logs si une anomalie apparaît,
- activer les options de verrouillage automatique.

## Paramètres utiles

La page Paramètres permet notamment de régler :

- affichage et thème,
- options liées à la sécurité,
- comportements de l'application,
- options d'intégration extension.
