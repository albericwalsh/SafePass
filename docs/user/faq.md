# FAQ

## Où sont publiées les nouvelles versions ?

Les versions sont publiées dans les Releases du dépôt public SafePass.

## Dois-je réinstaller complètement à chaque version ?

Non. Fermez SafePass, remplacez les fichiers par ceux de la nouvelle version, puis relancez.

## L'extension fonctionne-t-elle sans SafePass lancé ?

Non. L'extension communique avec le backend local SafePass.

## Que faire si l'extension ne propose aucun compte ?

Vérifier que :

- SafePass est lancé,
- le site/domain est correctement enregistré dans vos entrées,
- le jeton extension est valide si activé.

## Puis-je utiliser SafePass sur plusieurs machines ?

Oui, mais vous devez gérer la synchronisation de vos données de manière sécurisée selon votre environnement.
