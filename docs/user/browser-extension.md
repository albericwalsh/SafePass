# Extension navigateur

L'extension SafePass aide à récupérer et remplir vos identifiants sur les sites web.

## Installer l'extension (mode non-store)

1. Décompresser `SafePass-extension-X.Y.Z.zip`.
2. Ouvrir la page des extensions du navigateur Chromium (`chrome://extensions`).
3. Activer le mode développeur.
4. Choisir Charger l'extension non empaquetée.
5. Sélectionner le dossier de l'extension.

## Préparer la connexion avec SafePass

1. Lancer SafePass localement.
2. Ouvrir la popup de l'extension.
3. Saisir le jeton extension si votre configuration l'exige.
4. Vérifier l'état de connexion.

## Utilisation de base

1. Ouvrir une page de connexion.
2. Clic droit dans un champ éditable.
3. Sélectionner l'action SafePass.
4. Choisir le compte à utiliser.

L'extension remplit ensuite les champs détectés.

## Bonnes pratiques

- ne jamais partager le jeton extension,
- vérifier régulièrement l'état de connexion,
- garder l'application et l'extension à jour.
