# Démarrage rapide

Ce guide vous aide à démarrer SafePass en quelques minutes.

## 1) Ouvrir SafePass

Lancer `SafePass.exe`.

Attendre l'ouverture de l'interface principale.

## 2) Vérifier la configuration de base

Dans Paramètres :

- vérifier la langue,
- vérifier le thème,
- vérifier les options de sécurité (verrouillage automatique, etc.).

## 3) Créer une première entrée

Depuis l'écran principal :

1. cliquer sur Ajouter,
2. renseigner au minimum le service/site, l'identifiant et le mot de passe,
3. enregistrer.

## 4) Tester la recherche

Utiliser la recherche pour retrouver rapidement votre nouvelle entrée.

## 5) Bonnes pratiques immédiates

- utiliser des mots de passe uniques par service,
- activer le verrouillage automatique,
- verrouiller l'ordinateur quand vous vous absentez,
- conserver SafePass à jour.
